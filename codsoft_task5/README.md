Project Title:

Contact Management System

Overview:

This application enables users to store and manage personal or professional contact information. It supports adding, searching, updating, viewing, and deleting contacts efficiently. The project emphasizes structured data handling and menu-driven user interaction.

Features:

Add new contact details

View all stored contacts

Search for specific contacts

Update existing entries

Remove unwanted contacts

How to Run:

Run contact_book.py using Python in any terminal window.

Technologies Used:

Python and simple structured data handling techniques.

Project Structure:

This folder contains a README file, the contact_book.py script, and an output screenshot showing usage.

Input/Output Example:

Users provide a name, phone number, and email address. The program validates and stores the contact information.
