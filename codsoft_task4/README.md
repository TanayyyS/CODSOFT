TASK 4 – ROCK-PAPER-SCISSORS GAME:
Project Title:

Rock-Paper-Scissors Terminal Game

Overview:

This project is an interactive game where a user plays Rock‑Paper‑Scissors against the computer. The computer selects a move at random, and the program displays the winner of each round. The game keeps score until the user chooses to exit.

Features:

Generates random computer choices

Displays round results clearly

Maintains a scoreboard for the session

Provides a clean exit when user selects "no"

How to Run:

Run rock_paper_scissors.py from any Python-enabled terminal. Follow the prompts to choose your move.

Technologies Used:

Python and the random module for generating unpredictable computer choices.

Project Structure:

This folder includes a README, the rock_paper_scissors.py script, and an output screenshot demonstrating game flow.

Input/Output Example:

Users choose rock, paper, or scissors. The system responds with the computer’s move and the round result.
